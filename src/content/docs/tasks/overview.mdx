Задачи важнейшая часть системы Accept.
Они являются строительными блоками для [уроков](/assignments/overview), [турниров](/tournaments/overview) и [курсов](/courses/overview).
Учителя могу создавать, редактировать и удалять **любые** задачи внутри своей организации.

## Список задач
Здесь отображаются все доступные вам задачи. Так же с этой страницы учителя имеют доступ к добавлению задач.

Вы можете осуществлять поиск по названию, логину автора, тегам. Так же вы можете отключать или включать отображение некоторых столбцов.

![Страница со списком задач](../../../assets/task/overview/step-2.png)

## Страница задачи
Страница задачи состоит из трех вкладок: [описание](#описание), [отправка](#отправка), [результаты](#результаты).

Учителю доступно редактирование и удаление задачи, а так же редактирование тестов задачи.

Если задача относиться к [турниру](/tournaments/overview) или [уроку](/assignments/overview), то появляются
дополнительные элементы интерфейса: таймер до окночания турнира/урока и панель с результатами по задачам турнира/урока и ссылками быстрого перехода.

:::note
Неавторизованный пользователь видит исключительно вкладку [описание](#описание).
:::

### Описание
На этой вкладке пользователю доступна информация необходимая для решения задачи.
Страница состоит из нескольких разделов:
1. Заголовок, теги, логин автора, ограничения по памяти и времени
2. Описание задачи
3. Примеры входных и соответсвующих им выходных данных
4. (Опцианально) Примечание

![Описание задачи](../../../assets/task/description/step-1.png)

### Отправка
На этой вкладке пользователь может загрузить файл с решением или написать решение в соответсвующем поле,
отправить [попытку](/tasks/attempts/overview).

На выбор доступен ряд языков программирования: Python, C++, Pascal, C#, Java и [т.д.](/tasks/attempts/languages).

![Вкладка отправки](../../../assets/task/send/step-1.png)

### Результаты
На этой вкладке пользователь может просматривать результаты своих [посылок](/tasks/attempts/overview).
Нажатием на дату отправки попытки пользователь может перейти на [страницу попытки](/tasks/attempts/overview).

:::tip
(1) Со списком доступных языков можно ознакомиться на [странице](/tasks/attempts/additional#поддерживаемые-языки).

(2) Со списком вердиктов можно ознакомиться на [странице](/tasks/attempts/additional#вердикты-попыток).
:::
![Вкладка результатов](../../../assets/task/results/step-2.png)

## Доабвление и редактирование
Добавление и редактирование доступны на [странице](/tasks/add-edit).

## Удаление задачи
Учителям доступно удаление задачи.
При удалении задачи она будет удалени из всех [уроков](/assignments/overview), [турниров](/tournaments/overview) и [курсов](/courses/overview), в которых она использовалась

:::note
Если задача была единственной в уроке/турнире/курсе, её удаление НЕ приведет к удалению урока/турнира/курса.
:::

:::danger
Будьте аккуратны, удаление происходит безвозвратно!
:::
